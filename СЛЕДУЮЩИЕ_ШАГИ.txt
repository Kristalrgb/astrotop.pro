ШАГ 1: ВВЕСТИ ПАРОЛЬ
────────────────────
1. Введите пароль от GitHub в поле "Password"
2. Нажмите кнопку "Sign in" (зеленая)

ШАГ 2: РАЗРЕШИТЬ ДОСТУП RAILWAY
─────────────────────────────────
GitHub спросит:
"Authorize Railway App?"

→ Нажмите "Authorize railway"

Также выберите:
→ "All repositories" (все репозитории)
ИЛИ
→ "Only select repositories" → выберите "astrotop.pro"

ШАГ 3: ВЫБРАТЬ РЕПОЗИТОРИЙ
───────────────────────────
Railway покажет список ваших репозиториев:

1. Найдите: "astrotop.pro" или "Kristalrgb/astrotop.pro"
2. Нажмите на него
3. Railway начнет деплой автоматически

ШАГ 4: НАСТРОИТЬ ROOT DIRECTORY
────────────────────────────────
После начала деплоя (через 30-60 секунд):

1. В Railway нажмите на созданный проект
2. Откройте вкладку "Settings"
3. Прокрутите до "Build & Deploy Settings"
4. Найдите поле "Root Directory"
5. Введите: server
6. Нажмите "Save" или "Update"

Railway автоматически перезапустит деплой.

ШАГ 5: ДОБАВИТЬ ПЕРЕМЕННУЮ ОКРУЖЕНИЯ
─────────────────────────────────────
1. В том же разделе Settings найдите "Variables"
2. Нажмите "New Variable"
3. Заполните:
   • Key: FRONTEND_URL
   • Value: https://astrotoppro.vercel.app
4. Нажмите "Add"

ШАГ 6: ПОЛУЧИТЬ URL БЭКЕНДА
────────────────────────────
1. Откройте вкладку "Deployments"
2. Дождитесь зеленой галочки ✅ (это означает успешный деплой)
3. Откройте "Settings" → "Networking"
4. Найдите раздел "Public Networking"
5. Нажмите "Generate Domain"
6. СКОПИРУЙТЕ полученный URL!

   Пример: https://astrotop-backend-production.up.railway.app

⚠️ ВАЖНО: Сохраните этот URL!

ШАГ 7: ОБНОВИТЬ VERCEL
───────────────────────
1. Откройте новую вкладку: https://vercel.com/dashboard
2. Найдите проект "astrotop.pro"
3. Откройте "Settings" → "Environment Variables"
4. Нажмите "Add New"

   Добавьте ПЕРВУЮ переменную:
   • Key: VITE_API_URL
   • Value: https://ваш-бэкенд.railway.app
     (вставьте URL из шага 6)
   • Environment: 
     ✅ Production
     ✅ Preview  
     ✅ Development
   • Нажмите "Save"

   Добавьте ВТОРУЮ переменную:
   • Key: VITE_SOCKET_URL
   • Value: https://ваш-бэкенд.railway.app
     (тот же URL)
   • Environment:
     ✅ Production
     ✅ Preview
     ✅ Development
   • Нажмите "Save"

ШАГ 8: ПЕРЕСОБРАТЬ ФРОНТЕНД
────────────────────────────
1. В Vercel откройте вкладку "Deployments"
2. Найдите последний деплой (самый верхний)
3. Нажмите на "..." (три точки) справа
4. Выберите "Redeploy"
5. Подтвердите пересборку
6. Дождитесь завершения (1-2 минуты)

✅ ГОТОВО!
