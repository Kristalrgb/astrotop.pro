   ИСПРАВЛЕНИЕ ОШИБКИ ДЕПЛОЯ
═══════════════════════════════════════════════════════

ОШИБКА: "vite: command not found"

ПРОБЛЕМА: Build Command должен быть "npm run build"
          вместо "vite build"

═══════════════════════════════════════════════════════
   КАК ИСПРАВИТЬ:
═══════════════════════════════════════════════════════

1. На странице с ошибкой нажмите "Go to Project"
   (или откройте настройки проекта в Vercel)

2. Перейдите в раздел "Settings"

3. Найдите раздел "Build & Development Settings"

4. Найдите поле "Build Command"

5. Измените:
   ❌ БЫЛО: vite build
   ✅ ДОЛЖНО БЫТЬ: npm run build

6. Нажмите "Save"

7. Вернитесь в "Deployments"

8. Нажмите на последний деплой (с ошибкой)

9. Нажмите три точки (⋮) → "Redeploy"

10. Подождите 2-3 минуты

11. Готово! Должно заработать!
