✅ Railway уже открыт в браузере!
✅ Страница входа в GitHub открыта!

📋 ЧТО ДЕЛАТЬ СЕЙЧАС:

ШАГ 1: ВОЙТИ В GITHUB
──────────────────────
На открытой странице GitHub:

1. Введите ваш Username или Email
2. Введите ваш Password
3. Нажмите "Sign in"

Если есть 2FA → введите код

ШАГ 2: РАЗРЕШИТЬ ДОСТУП RAILWAY
─────────────────────────────────
GitHub спросит:
→ "Authorize Railway App?"
→ Нажмите "Authorize railway"

Также выберите доступ к репозиториям:
→ "All repositories" или
→ "Only select repositories" → выберите astrotop.pro

ШАГ 3: ВЫБРАТЬ РЕПОЗИТОРИЙ
───────────────────────────
Railway покажет список репозиториев:

1. Найдите: "astrotop.pro"
2. Нажмите на него
3. Railway начнет деплой

ШАГ 4: НАСТРОИТЬ ROOT DIRECTORY
────────────────────────────────
После начала деплоя:

1. Откройте Settings (настройки)
2. Прокрутите до "Build & Deploy Settings"
3. Найдите "Root Directory"
4. Введите: server
5. Нажмите Save

Railway перезапустит деплой.

ШАГ 5: ДОБАВИТЬ ПЕРЕМЕННУЮ
───────────────────────────
1. В Settings → "Variables"
2. Нажмите "New Variable"
3. Добавьте:
   • Name: FRONTEND_URL
   • Value: https://astrotoppro.vercel.app
4. Нажмите "Add"

ШАГ 6: ПОЛУЧИТЬ URL БЭКЕНДА
────────────────────────────
1. Откройте "Deployments"
2. Дождитесь зеленой галочки ✅
3. Settings → "Networking"
4. "Public Networking" → "Generate Domain"
5. СКОПИРУЙТЕ URL (например: xxx.up.railway.app)

⚠️ СОХРАНИТЕ ЭТОТ URL!

ШАГ 7: ОБНОВИТЬ VERCEL
───────────────────────
1. Откройте: https://vercel.com/dashboard
2. Найдите проект: "astrotop.pro"
3. Settings → "Environment Variables"
4. Добавьте 2 переменные:

   Переменная 1:
   • Key: VITE_API_URL
   • Value: https://ваш-бэкенд.railway.app
   • Environment: ✅ Production ✅ Preview ✅ Development

   Переменная 2:
   • Key: VITE_SOCKET_URL
   • Value: https://ваш-бэкенд.railway.app
   • Environment: ✅ Production ✅ Preview ✅ Development

5. Нажмите "Save"

ШАГ 8: ПЕРЕСОБРАТЬ ФРОНТЕНД
────────────────────────────
1. В Vercel → "Deployments"
2. Нажмите "..." на последнем деплое
3. Выберите "Redeploy"
4. Дождитесь завершения (1-2 минуты)

✅ ГОТОВО! Проверьте публикацию новости на сайте.

═══════════════════════════════════════════════════════════════

🎯 РЕЗУЛЬТАТ: Публикация новостей будет работать!
